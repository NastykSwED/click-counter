# Simple Click Counter App

This is my first React App using Functional Components and Hooks

## Run Locally

Clone the project

```bash
  git clone https://github.com/NastykSwED/click-counter
```

Go to the project directory

```bash
  cd click-counter
```

Install dependencies

```bash
 pnpm install
```

Start the server

```bash
  pnpm run dev
```
